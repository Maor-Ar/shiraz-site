 The .global-leaves container was position: fixed covering the entire viewport with z-index: 2, while the content <section> elements had position: relative but no z-index (defaults to auto/0). This meant the fixed leaf overlay was painting on top of ALL content, making everything invisible behind it.


✅ The Fix

Added z-index: 5 to all section, main, footer, header elements — this pushes all content above the decorative overlays
Lowered .global-leaves z-index from 2 → 1 — leaves now sit between the particles canvas (z-index: 0) and content (z-index: 5)
Set nav z-index to 100 — ensures navigation stays above everything
Set cursor follower z-index to 50 — visible above content but below nav
Set scroll progress z-index to 200 — always visible at top
Set loader z-index to 9999 — covers everything during load
